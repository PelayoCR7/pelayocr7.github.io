Web hosting

El web hosting es un servicio que provee espacio en servidores para almacenar y hacer accesibles los archivos de un sitio web en internet. Proporciona el espacio y recursos necesarios para que un sitio web esté disponible en línea.
En mi caso he utilizado GitHub que es una plataforma en línea que permite a desarrolladores almacenar, compartir y colaborar en proyectos de software.
