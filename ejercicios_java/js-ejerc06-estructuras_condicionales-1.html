Estructuras condicionales

Lo que se hace en este ejercicio es un script en el que se pregunta al usuario si quiere jugar a un juego, si responde que sí entonces se le preguntará por una fruta. Debido a que hay muchas frutas, se han escogido las más populares, por lo que se ha hecho un código esperando que el usuario diga una de esas frutas, sino se responderá que no hay de esa clase de fruta por el momento. Al decir una fruta, se dirá el precio de cada una. Ya después de eso, se le preguntará al usuario si quiere algo más, si dice que sí ,con el if, se le dirá un alert de que no hay tiempo y si dice que no pues se le agradecerá su consulta. Por último hay un else que está encadenado al primer if de que si quería jugar a un juego, gracias este else si el usuario dice que no pues s ele dirá que es una pena que no quiera jugar.
